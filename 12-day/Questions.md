4.¿Qué hace el operador ...rest en una desestructuración de arrays?
En una desestructuración de arrays, el operador ...rest (conocido como "parámetro rest" o "patrón rest") se utiliza para recopilar los elementos restantes del array que no han sido asignados explícitamente a otras variables. Estos elementos se agrupan en un nuevo array.
5.¿Qué hace la asignación por desestructuración en un array?
permite "desempaquetar" o extraer los valores de un array y asignarlos directamente a variables individuales. Es una forma concisa de tomar elementos de un array y darles nombres de variables específicos.
